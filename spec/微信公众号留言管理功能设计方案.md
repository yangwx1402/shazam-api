# 微信公众号留言管理功能设计方案

## 1. 项目概述

### 1.1 背景
实现微信公众号留言评论的管理功能，通过 Java SDK 封装微信开放平台的留言管理 API，包括打开/关闭评论、查看评论、管理评论（精选、删除、回复）等功能。

### 1.2 功能范围
- **评论开关**：打开/关闭已群发文章的评论功能
- **评论查询**：查看指定文章的评论数据
- **评论管理**：标记精选、取消精选、删除评论
- **回复管理**：回复评论、删除回复

### 1.3 参考文档
- [微信开放平台 - 打开已群发文章评论](https://developers.weixin.qq.com/doc/subscription/api/leaving/api_openarticlecomment.html)
- [微信开放平台 - 留言管理](https://developers.weixin.qq.com/doc/offiaccount/Comments/Comment_data_interface.html)

### 3.2 API 接口设计

#### 3.2.1 留言管理 API (CommentApi)

```java
/**
 * 留言管理接口
 */
public interface CommentApi extends WechatApi {

    /**
     * 打开已群发文章评论
     *
     * @param request 打开评论请求
     * @return 操作结果
     * @throws WechatApiException API 调用异常
     */
    CommentOpenResponse openComment(CommentOpenRequest request);

    /**
     * 关闭已群发文章评论
     *
     * @param request 关闭评论请求
     * @return 操作结果
     * @throws WechatApiException API 调用异常
     */
    CommentCloseResponse closeComment(CommentCloseRequest request);

    /**
     * 获取指定文章的评论数据
     *
     * @param request 获取评论请求
     * @return 评论列表
     * @throws WechatApiException API 调用异常
     */
    CommentGetResponse getComments(CommentGetRequest request);

    /**
     * 将评论标记为精选
     *
     * @param request 标记精选请求
     * @return 操作结果
     * @throws WechatApiException API 调用异常
     */
    CommentMarkResponse markComment(CommentMarkRequest request);

    /**
     * 将评论取消精选
     *
     * @param request 取消精选请求
     * @return 操作结果
     * @throws WechatApiException API 调用异常
     */
    CommentUnmarkResponse unmarkComment(CommentUnmarkRequest request);

    /**
     * 删除评论
     *
     * @param request 删除评论请求
     * @return 操作结果
     * @throws WechatApiException API 调用异常
     */
    CommentDeleteResponse deleteComment(CommentDeleteRequest request);

    /**
     * 回复评论
     *
     * @param request 回复评论请求
     * @return 操作结果
     * @throws WechatApiException API 调用异常
     */
    CommentReplyResponse replyComment(CommentReplyRequest request);

    /**
     * 删除回复
     *
     * @param request 删除回复请求
     * @return 操作结果
     * @throws WechatApiException API 调用异常
     */
    CommentDeleteResponse deleteReply(CommentDeleteReplyRequest request);
}
```

## 4. 使用示例

### 4.1 快速开始

```java
import com.shazam.wechat.sdk.WechatClient;
import com.shazam.wechat.sdk.WechatConfig;
import com.shazam.wechat.sdk.model.request.*;
import com.shazam.wechat.sdk.model.response.*;

public class WechatCommentExample {
    public static void main(String[] args) {
        // 1. 创建客户端
        WechatConfig config = new WechatConfig.Builder()
            .appId("your_app_id")
            .appSecret("your_app_secret")
            .build();
        WechatClient client = new WechatClient(config);

        try {
            // 2. 获取 access_token
            client.auth().getStableAccessToken();

            // 3. 打开评论
            CommentOpenRequest openRequest = new CommentOpenRequest.Builder()
                .msgDataId(1234567890L)
                .index(0)  // 第一篇图文
                .build();

            CommentOpenResponse openResponse = client.comment().openComment(openRequest);
            if (openResponse.isSuccess()) {
                System.out.println("评论功能已打开");
            }

            // 4. 查看评论
            CommentGetRequest getRequest = new CommentGetRequest.Builder()
                .msgDataId(1234567890L)
                .index(0)
                .begin(0)
                .count(20)
                .build();

            CommentGetResponse getResponse = client.comment().getComments(getRequest);
            if (getResponse.isSuccess()) {
                System.out.println("评论总数：" + getResponse.getTotal());
                for (CommentInfo comment : getResponse.getCommentList()) {
                    System.out.println("评论 ID: " + comment.getCommentId());
                    System.out.println("用户：" + comment.getOpenid());
                    System.out.println("内容：" + comment.getContent());
                    System.out.println("是否精选：" + (comment.getIsTop() == 1 ? "是" : "否"));
                }
            }

            // 5. 标记精选
            CommentMarkRequest markRequest = new CommentMarkRequest.Builder()
                .msgDataId(1234567890L)
                .commentId(987654321L)
                .build();

            CommentMarkResponse markResponse = client.comment().markComment(markRequest);
            if (markResponse.isSuccess()) {
                System.out.println("评论已标记为精选");
            }

        } catch (WechatApiException e) {
            System.err.println("操作失败：" + e.getErrorCode() + " - " + e.getErrorMsg());
        } finally {
            client.shutdown();
        }
    }
}
```

### 4.2 完整评论管理流程

```java
// 打开评论

    .msgDataId(1234567890L)
    .index(0)
    .build();

// 管理评论

        .msgDataId(1234567890L)
        .commentId(comment.getCommentId())
        .build();
    client.comment().markComment(markRequest);

    // 回复评论
    CommentReplyRequest replyRequest = new CommentReplyRequest.Builder()
        .msgDataId(1234567890L)
        .commentId(comment.getCommentId())
        .content("感谢您的评论！")

    client.comment().replyComment(replyRequest);

    // 删除不良评论
    if (isBadComment(comment)) {

}
```

## 5. 注意事项

### 5.1 权限要求

- 只有已微信认证的服务号才支持留言功能
- 订阅号部分支持留言（需满足特定条件）
- 新开通账号可能需要申请留言权限

### 5.2 使用限制

| 限制项 | 说明 |
|--------|------|
| 评论数量 | 每篇文章最多显示 50 条评论 |
| 回复长度 | 回复内容不超过 200 字 |
| 调用频率 | 天级别频率限制（具体数值未公开） |
| 敏感词过滤 | 系统自动过滤敏感词 |

### 5.3 评论状态

| 状态 | is_top | is_deleted | 说明 |
|------|--------|------------|------|
| 正常 | 0 | 0 | 普通评论 |
| 精选 | 1 | 0 | 精选评论（显示在文章顶部） |
| 删除 | 0/1 | 1 | 已删除评论 |

### 5.4 群发后操作

- 群发后默认关闭评论
- 需要手动调用 `openComment` 打开评论
- 关闭评论后，用户无法继续评论，但已有评论保留

### 5.5 msg_data_id 获取

- `msg_data_id` 来自群发接口的返回值
- 可通过 `get_current_selfmenu_info` 等接口获取
- 多图文消息时，使用 `index` 指定具体文章
