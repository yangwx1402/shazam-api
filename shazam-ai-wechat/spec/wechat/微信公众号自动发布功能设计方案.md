# 微信公众号自动发布功能设计方案

## 1. 项目概述

### 1.1 背景
实现微信公众号内容的自动发布功能，通过 Java SDK 封装微信开放平台的草稿箱管理和发布能力 API，简化开发流程。

### 1.2 功能范围
- **素材管理**：上传图片等临时素材
- **草稿管理**：创建、获取、删除、查询草稿
- **发布管理**：提交发布、查询发布状态

### 1.3 参考文档
- [微信开放平台 - 草稿箱管理](https://developers.weixin.qq.com/doc/subscription/api/draftbox/draftmanage/api_draft_add.html)
- [微信开放平台 - 发布能力](https://developers.weixin.qq.com/doc/offiaccount/Draft_Box/Add_Draft.html)

### 3.2 API 接口设计

#### 3.2.1 素材管理 API (MaterialApi)

```java
/**
 * 素材管理接口
 */
public interface MaterialApi extends WechatApi {

    /**
     * 上传图片（获取图片 URL）
     * 用于图文消息封面或内容中的图片
     *
     * @param imageFile 图片文件
     * @return 上传结果，包含图片 URL
     * @throws WechatApiException API 调用异常
     */
    UploadImageResponse uploadImage(File imageFile);

    /**
     * 上传图片（指定文件路径）
     *
     * @param imagePath 图片文件路径
     * @return 上传结果，包含图片 URL
     * @throws WechatApiException API 调用异常
     */
    UploadImageResponse uploadImage(String imagePath);
}
```

#### 3.2.2 草稿管理 API (DraftApi)

```java
/**
 * 草稿管理接口
 */
public interface DraftApi extends WechatApi {

    /**
     * 新增草稿
     *
     * @param request 草稿请求
     * @return 新增结果，包含 media_id
     * @throws WechatApiException API 调用异常
     */
    DraftAddResponse addDraft(DraftAddRequest request);

    /**
     * 获取草稿详情
     *
     * @param mediaId 草稿 media_id
     * @return 草稿详情
     * @throws WechatApiException API 调用异常
     */
    DraftGetResponse getDraft(String mediaId);

    /**
     * 删除草稿
     *
     * @param mediaId 草稿 media_id
     * @return 删除结果
     * @throws WechatApiException API 调用异常
     */
    DraftDeleteResponse deleteDraft(String mediaId);

    /**
     * 更新草稿
     *
     * @param request 更新请求
     * @return 更新结果
     * @throws WechatApiException API 调用异常
     */
    DraftUpdateResponse updateDraft(DraftUpdateRequest request);

    /**
     * 获取草稿列表
     *
     * @param offset 从全部素材的哪一项开始，默认为 0
     * @param count 素材数量，默认 20，最大 20
     * @return 草稿列表
     * @throws WechatApiException API 调用异常
     */
    DraftBatchGetResponse batchGetDrafts(int offset, int count);

    /**
     * 获取草稿总数
     *
     * @return 草稿总数
     * @throws WechatApiException API 调用异常
     */
    DraftCountResponse countDrafts();
}
```

## 4. 使用示例

### 4.1 快速开始

```java
import com.shazam.wechat.sdk.WechatClient;
import com.shazam.wechat.sdk.WechatConfig;
import com.shazam.wechat.sdk.model.request.*;
import com.shazam.wechat.sdk.model.response.*;

public class WechatPublishExample {
    public static void main(String[] args) {
        // 1. 创建客户端
        WechatConfig config = new WechatConfig.Builder()
            .appId("your_app_id")
            .appSecret("your_app_secret")
            .build();
        WechatClient client = new WechatClient(config);

        try {
            // 2. 上传封面图片
            File coverImage = new File("/path/to/cover.jpg");
            UploadImageResponse imgResp = client.material().uploadImage(coverImage);
            String thumbMediaId = imgResp.getUrl();

            // 3. 创建草稿
            DraftArticle article = new DraftArticle.Builder()
                .title("我的文章标题")
                .author("作者名")
                .content("<p>文章 HTML 内容...</p>")
                .thumbMediaId(thumbMediaId)
                .needOpenComment(1)
                .build();

            DraftAddRequest draftRequest = new DraftAddRequest.Builder()
                .addArticle(article)
                .build();

            DraftAddResponse draftResp = client.draft().addDraft(draftRequest);
            String mediaId = draftResp.getMediaId();
            System.out.println("草稿创建成功，media_id: " + mediaId);

            // 4. 提交发布
            PublishSubmitRequest publishRequest = new PublishSubmitRequest.Builder()
                .mediaId(mediaId)
                .build();

            PublishSubmitResponse publishResp = client.publish().submitPublish(publishRequest);
            String publishId = publishResp.getPublishId();
            System.out.println("发布提交成功，publish_id: " + publishId);

            // 5. 查询发布状态
            Thread.sleep(2000);  // 等待发布完成
            PublishGetResponse statusResp = client.publish().getPublishStatus(publishId);

            if (statusResp.getPublishStatus() == 0) {
                System.out.println("发布成功！文章链接：" + statusResp.getArticleUrl());
            } else if (statusResp.getPublishStatus() == 1) {
                System.out.println("发布中...");
            } else {
                System.out.println("发布失败");
            }

        } catch (WechatApiException e) {
            System.err.println("API 调用失败：" + e.getErrorCode() + " - " + e.getErrorMsg());
        } catch (InterruptedException e) {
            Thread.currentThread().interrupt();
        } finally {
            client.shutdown();
        }
    }
}
```

## 5. 错误码参考

| 错误码 | 说明 | 解决方案 |
|-------|------|---------|
| 40001 | access_token 无效 | 检查 token 是否过期 |
| 40004 | 不合法的媒体文件类型 | 检查文件类型是否符合要求 |
| 40005 | 文件大小超出限制 | 压缩文件或检查限制 |
| 40007 | 草稿 media_id 不存在 | 检查 media_id 是否正确 |
| 40013 | AppID 不合法 | 检查 AppID 配置 |
| 40015 | 草稿内容不合法 | 检查 HTML 内容和格式 |
| 40125 | AppSecret 无效 | 检查 AppSecret 配置 |
| 40164 | IP 不在白名单 | 在微信后台添加 IP 白名单 |
| 45009 | 超过频率限制 | 降低调用频率或调用 clear_quota |
| 53404 | 账号被限制带货能力 | 删除商品后重试 |

## 6. 注意事项

### 6.1 Access Token
- 所有接口都需要有效的 access_token
- SDK 会自动管理 token 缓存（默认 2 小时）
- 稳定版 token 不受 IP 白名单限制

### 6.2 图片上传
- 图文消息的图片必须通过微信接口上传
- 不能直接使用外部图片链接
- 图片大小限制：不超过 10MB
- 支持格式：JPG、PNG、GIF

### 6.3 内容规范
- 标题不超过 32 个字
- 摘要不超过 128 个字（仅单图文有效）
- 内容小于 2 万字符
- HTML 内容需符合微信规范

### 6.4 发布限制
- 公众号每天发布次数有限（通常 1-2 次）
- 草稿箱容量有限，及时清理无用草稿
- 发布后素材从草稿箱移除

### 6.5 频率限制
- 各接口有不同的调用频率限制
- 超限后可调用 `clear_quota` 恢复
- 建议使用 SDK 的缓存机制
